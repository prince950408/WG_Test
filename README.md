# WG_Test

## Installation
Before you begin, ensure you have met the following requirements:
Node.js 22 installed

## Setup
Clone the repository:

```
git clone https://github.com/prince950408/WG_Test.git
cd WG_Test
npm install
```

## Usage
Start server:
```
npm start
```
Will run in http://localhost:3000/

## Test
```
npm test
```
